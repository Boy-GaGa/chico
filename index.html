¿Que es Chico-UI?

Chico-UI es un framework visual para la web montado sobre jQuery, es proyecto opensource de MercadoLibre. Su Diseño es simple y esta desarrollado con tecnologías abiertas de la web: HTML, JavaScript y CSS... y un poco de PHP.

Utiliza algunas características de HTML5 y CSS3, provee métodos de compatibilidad en navegadores de grado-c con jQuery y otros artilugios.

Chico-UI agrupa los patrones y componentes mas comunes con el objetivo de agilizar la producción de interfaces en los productos de MercadoLibre. Podés verlo en github. Esta listo para usar!

Download
