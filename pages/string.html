¿Cómo se usa?

El objecto String es una interface del objecto Watchers donde se pueden definir algunas validaciones preformateadas de cadenas de caracteres. Recibe como argumento un objeto de propiedades que definen los tipos de validaciones y algunos aspecto de su comportamiento como los callbacks.

Ejemplo del objeto de configuración:

        {
            text: boolean,
            email: boolean,
            url: boolean,
            maxLength: int,
            minLength: int,
            callbacks: {
                validate: function() {},
                reset: function() {}
            }
        }

Realizando una selección con jQuery podemos asignarle una configuración de la siguiente forma:

        $("#Foobar").string({ // Check for a valid email format smaller than 20 chars
            email: true,
            maxLength: 20
        });

Además, el componente soporta algunos shorthands que se pueden concatenar con el método and() facilitando su instanciación con otras validaciones de distintos tipos, por ejemplo: 

        $("#Foobar").email().and().maxLength(20); // Same example

Cada vez que se instancia un componente de la familia Watchers éste retorna un objecto público de control que nos brinda información sobre la instancia y nos permite ejecutar algunos métodos públicos.

        var foo = $("#Foobar").email().and().maxLength(20); // Same example
        
        // returns
        foo = {
            uid: 121,                         //  Global number instance
            element: HTMLInputElement,        //  Input DOM Interface
            type: "watcher",                  //  Component Type
            status: boolean,                  //  Status
            reference: HTMLInputElement,      //  Positioner reference
            parent: HTMLFormElement,          //  Form parent
            validations: {                    //  Validation Map
                email: true,                  
                maxLength: 20                 
            },
            messages:{},                      //  Messages Map
            and(),                            //  Concatenate method 
            reset(),                          //  Reset the value
            validate(),                       //  Execute the validations
        }

En la configuración de los campos podemos conbinar con el método and() con otras interfaces del componente Watchers como la validación de campo requerido Required.

El Javascript para iniciarlo es:

    $(".c1").forms();
    $("#c1_email").email().and().required();
    $("#c1_url").url().and().required();
   	   
/*  Otra alternativa es agrupar validaciones requeridas en una instrucción:
    $("#c1_email").email();
    $("#c1_url").url();
    $(".c1 input").required();
