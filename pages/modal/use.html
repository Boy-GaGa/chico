Modal Window

El modal es disparado por un link o botón del tipo submit (dentro de un form). Esto puede ser en el evento click o bien cunado uno lo desee utilizando su método público.

Carga su contenido tanto por ajax como por parametro. Al configurarse para que lo cargue por ajax, toma por default la url del elemento HTML que lo incia (en caso de que sea un link, levanta el href; si es un botón de tipo submit, levatan el action del form).

Por otra parte, el modal ofrece una clase de CSS closeModal la cual se la puede agregar a cualquier elemento HTML para que cierre el modal.

Ejemplo del objeto de configuración:

		{
			content: string,
			ajax: boolean | string,
			callbacks: {
				show: function() {},
				hide: function() {}
			}
		}

Realizando una selección con jQuery podemos asignarle una configuración de la siguiente forma:

        $("#Foobar").modal({
        	ajax: true,        	
        });

Cada vez que se instancia un componente Modal Window retorna un objecto público de control que nos brinda información sobre la instancia y nos permite ejecutar algunos métodos públicos.

		var foo = $("#Foobar").modal();
        
		// returns
		foo = {
			uid: 121,                         //  Global number instance
			element: HTMLInputElement,        //  Input DOM Interface
			type: "modal",                    //  Component Type
			content: string,				  //  Get content
			position(),					  	  //  Refresh position
			show(),						  	  //  Show modal
			hide()						  	  //  Hide modal
		}
