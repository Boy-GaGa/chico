How to install

¿Cómo se usa?

Nota: Todos los archivos mencionados en las instrucciones se encuentran disponibles en el código fuente del proyecto.
Podés bajarlo desde la página de inicio.

Paso 1: Browser sniffing

Para lograr crossbrowsing visual en la capa de estilos CSS se encuentran pre-definidos los arreglos necesarios para errores comunes. Es necesario agregar un atributo class al elemento html, para lograrlo utilizamos los comentarios condicionales para IE:

Recomendaciones

Hay algunas concideraciones a tener en cuenta para no perjudicar el rendimiento de los navegadores en general.

  • Primero Browser Sniffer.
  • Agregar los estilos en la cabecera del documento.
  • Incluir los javascript al pie del documento.
  • Por último y de forma no instrusiva, agregar los scripts.
