Dropdown

El objeto Dropdown es un componente que agrupa una lista de links, los cuales disparan diferentes acciones. Puede recibir como argumento un objeto de propiedades para los callbacks.

Ejemplo del objeto de configuración:

		{
			onShow: function() {},			  // Callback executed on dropdown show
			onHide: function() {}			  // Callback executed on dropdown hide
		}

Realizando una selección con jQuery podemos asignarle una configuración de la siguiente forma:

        $("#Foobar").dropdown();

Cada vez que se instancia un componente Dropdown, éste retorna un objecto público de control que nos brinda información sobre la instancia y nos permite ejecutar algunos métodos públicos.

		var foo = $("#Foobar").dropdown();
        
		// returns
		foo = {
			uid: 121,                         //  Global number instance
			element: HTMLInputElement,        //  Input DOM Interface
			type: "dropdown",                 //  Component Type
			show(),						  	  //  Show dropdown
			hide()						  	  //  Hide dropdown
		}
