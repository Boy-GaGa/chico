Modal

Demo

Modal ejecutado en un link.

¿Cómo se usa?

El modal es disparado por un link o botón del tipo submit (dentro de un form). Esto puede ser en el evento click o bien cunado uno lo desee utilizando su método público.

Carga su contenido tanto por ajax como por parametro. Al configurarse para que lo cargue por ajax, toma por default la url del elemento HTML que lo incia (en caso de que sea un link, levanta el href; si es un botón de tipo submit, levatan el action del form).

Sintaxis

El HTML para utilizar este componente es:

			...
							
			ó

El javascript para iniciarlo es:

			$('.foo').modal();

Configuración

Parámetros de configuración que puede recibir este componente:

Ajax
boolean o string
true
Recibe este parametro para cargar su contenido por ajax utilizando la url del href o del action del form del botón.
string
Se le especifica una url en formato string de la cual cargará el contenido (puede ser tanto absoluta como relativa).
Content
string: Puede ser un selector de CSS, contenido HTML o texto plano.
callbacks
object
show
Recibe como parametros "show" y/o "hide". Son las funciones que se ejecutarán como respuesta del componente al mostrarse u ocultarse.
