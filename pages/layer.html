¿Cómo se usa?

El objecto Layer pertenece a la familia Floats (al igual que Modal y Tooltip) donde se pueden definir algunas validaciones preformateadas de cadenas de caracteres. Recibe como argumento un objeto de propiedades que definen los tipos de validaciones y algunos aspecto de su comportamiento como los callbacks.

Ejemplo del objeto de configuración:

        /* {
			content: string,
			ajax: string,
			event: string,
            callbacks: {
                show: function() {},
                hide: funciont() {}
            }
        } */

Realizando una selección con jQuery podemos asignarle una configuración de la siguiente forma:

        $("#Foobar").layer({
        	content: "
Lorem ipsum dolor sit amet, consectetur adipiscing elit
",
        	event: "click",
        });

Cada vez que se instancia un componente Layer retorna un objecto público de control que nos brinda información sobre la instancia y nos permite ejecutar algunos métodos públicos.

        var foo = $("#Foobar").layer() // Same example
        
        // returns
        foo = {
            uid: 121,                         //  Global number instance
            element: HTMLInputElement,        //  Input DOM Interface
            type: "watcher",                  //  Component Type
          	content:,						  //  Get content	
          	position:,						  //  Set position
          	show:,							  //  Set position
          	hide:							  //  Set position
        }

Casos de uso

Modal ejecutado en el click de un link con ajax

Ejecutar Modal.

El Javascript para iniciarlo es:

		$(".modal_link").modal(
			{
				ajax:true
			}
		);

Modal ejecutado en el click de un botón con ajax

El Javascript para iniciarlo es:

		$("#modal_btn").modal(
			{
				ajax:true
			}
		);
