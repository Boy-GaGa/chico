¿Cómo se usa?

El Tab Navigator se crea a partir de agregarle funcionalidad a un conjunto de anclas.

Sintaxis

El HTML para utilizar este componente es:

			

				
    
					
  • Tab1
    • 
					
  • Tab2
    • 
					
  • Tab3
    • 
				

				

					
...

					
...

					
...

El javascript para iniciarlo es:

			$(".foo").tabNavigator();

Configuración

Parámetros de configuración que puede recibir este componente:

callbacks
object
show
function: Se ejecutará cuando cada uno de los tabs sea mostrado.
hide
function: Se ejecutará cuando cada uno de los tabs deje de mostrarse.

Para cargar el contenido de un tab por ajax (en este caso, el tab 2), es necesario que el atributo "href" del link correspondiente, apunte a la URL de donde se va a obtener el contenido. El componente automáticamente creará el contenedor para alojar el resultado y lo vinculará con el tab en cuestión.

El javascript para iniciarlo es:

$(".demoTabNavigator").tabNavigator();
